== Description ==

This plug is used to show hot searches in blog.If anyone use wordpress intergrated search engineer or intergrated Google Adsense Search to search your post,it will recorde the key. And place the hot searches widget in the sidebar,you can see it.

== Installation ==

1. Upload the full directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `hot searches widget` in your sidebar.
